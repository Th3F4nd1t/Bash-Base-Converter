# BBC
To install
```bash
git clone https://github.com/Th3F4nd1t/Bash-Base-Converter
cd Bash-Base-Converter
sudo bash ./install.sh        # enter y when prompted

# optional but recommended removal of install materials
cd ..
rm -rf ./Bash-Base-Converter

`bbc <base_from> <base_to> number`

### Aliases
- `b2d`: base 2 to base 10
- `b2h`: base 2 to base 16
- `h2d`: base 16 to base 10
- `h2b`: base 16 to base 2
- `d2b`: base 10 to base 2
- `d2h`: base 10 to base 16
